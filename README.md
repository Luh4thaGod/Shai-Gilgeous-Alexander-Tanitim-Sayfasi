# Shai Gilgeous-Alexander Tanıtım Sayfası

Bu proje, Shai Gilgeous-Alexander hakkında hazırlanmış basit ve responsive bir web sayfasıdır. Sayfa HTML, CSS ve Bootstrap kullanılarak oluşturulmuştur.

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

## Sayfa İçeriği

Projede aşağıdaki bölümler bulunmaktadır:

- Responsive navbar
- Bootstrap carousel / slider
- Oyuncu bilgi kartları
- Öne çıkan özellikler bölümü
- Shai Gilgeous-Alexander hakkında tanıtım yazısı
- NBA oyuncu profiline yönlendiren buton
- Footer alanı
- Sosyal medya ikonları
- İletişim bilgileri

## Özellikler

- Mobil uyumlu tasarım
- Hamburger menü desteği
- Kayan görsel slider
- Bootstrap kart yapısı
- İkon destekli footer
- Harici bağlantı butonu
